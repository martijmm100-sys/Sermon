Deployment Guide

This static version is built to avoid the blank-page problem.

Files needed

This package only requires one live app file:

orthodox-sermon-forge-static/
  index.html
  README.md

Only index.html is required for the website to run.

Cloudflare Pages settings

Use these settings for this static package:

Framework preset: None
Build command: Leave blank
Build output directory: /

Do not use Vite settings for this standalone version.

Git Bash upload commands

cd ~/Desktop/orthodox-sermon-forge-static

git init
git add .
git commit -m "Deploy standalone Orthodox Sermon Forge"
git branch -M main
git remote add origin https://github.com/YOUR-GITHUB-USERNAME/orthodox-sermon-forge-static.git
git push -u origin main
Safety warning: do not upload API keys, passwords, .env files, proprietary company data, customer data, copyrighted books you do not have rights to publish, or confidential pastoral/family notes to GitHub.
Orthodox Sermon Forge is a study aid, not a substitute for Holy Scripture, the liturgical life of the Church, pastoral guidance, or verification against primary sources.
