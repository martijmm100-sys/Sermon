# Source Notice

This application is designed around:

- **Title:** *The Book of Enoch*
- **Translation:** R. H. Charles
- **Contributor / Introduction:** W. O. E. Oesterley
- **Public source:** Project Gutenberg eBook 77935
- **Project Gutenberg page:** <https://www.gutenberg.org/ebooks/77935>

Project Gutenberg lists the eBook as **public domain in the USA**.

## App handling

This GitHub Pages version is static. It does not ship a backend server. The app attempts to load the full Project Gutenberg text in the browser and parse it into chapters and verses. If browser security blocks this, the app still opens and provides:

- a bundled starter excerpt,
- an embedded Project Gutenberg source portal,
- a manual paste loader that lets the user build a complete native reader in the browser.

## Christian study note

1 Enoch is not canonical Scripture for most Christian traditions. This app treats it as ancient religious literature and provides devotional prompts that should be tested against Scripture, prayer, and sound church teaching.
